6Tools Plugin de base
=============

Fonctionnalités
-------
Ce plugin de base de développement intègre les fonctions de base pour :
- Creation des post-types et taxonomies
- Intégration de la librairy RW_Meta_Boxes
- Ajout de champs personnalisés dans l'admin d'un post (ici upload d'images, éditeur de texte)
- Gestion des templates : des templates de base sont présents dans le plugin pour les override il faut dupliquer le template dans le répertoire du thème en cours
- Fonctions d'activation, désactivation et desinstallation du plugin
- Ajout d'un shortcode
- Ajout d'un widget
- Ajout d'un menu de configuration du plugin dans le menu admin
